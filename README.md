# Console Raytracer
### Real-time CLI raytracer - 2 scenes

Preview:
[![asciicast](https://asciinema.org/a/dl8mqPk0S5hFL2Y1k3U7omHjN.png)](https://asciinema.org/a/dl8mqPk0S5hFL2Y1k3U7omHjN)

**Dependencies**
```
sudo apt install libncurses-dev
```

**Build**
```
make
```

**Run**
```
./raytracer
```
